# Image Hotspots

Tags: jQuery, javascript, php, css
Requires at least: 3.6.0
Tested up to: 6.9.1
License: GPL2

## Description

Add clickable hotspot markers to images using ACF Pro repeater fields with customisable tooltips and marker styles.

## Tested on 
* Firefox 
* Safari
* Chrome
* Opera
* MS Edge

## Website 
http://www.phildesigns.com/

## Prerequisite
* Requires ACF Pro

## Installation 
1. Upload ‘timage-hotspots’ to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the "THotSpots" custom post type and create "hotspot" posts there
4. Use the shortcode to embed the hotspot images on your posts and pages.
